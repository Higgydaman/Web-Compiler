HOW I STATED THIS PROJECT

# Get express
C:\Projects\Web Compiler> npm install express

# Get the express generator
C:\Projects\Web Compiler> npm install express-generator -g

# Generate the project
C:\Projects\Web Compiler> express --view=ejs web-comp

# Install the dependencies
C:\Projects\Web Compiler> cd web-comp
C:\Projects\Web Compiler\web-comp> npm install
